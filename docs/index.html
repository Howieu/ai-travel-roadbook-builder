Travel Roadbook Builder

把零散攻略变成产品级网页路书

面向自由行用户的 V0.2：把攻略链接、粘贴笔记、酒店/交通截图整理成带来源证据的 roadbook.json，再渲染成手机上可执行的产品向路书站点。

V0.2 工作流

普通网页通过 Agent Reach / Jina Reader 转成来源记录；小红书直读需要 OpenCLI 或 xiaohongshu-mcp 后端，不配置时回退到粘贴正文或截图；酒店和交通信息从用户截图里提取为行程约束。

来源证据

路书保留 sourceRecords 和 sourceIds，能区分网页、粘贴笔记、预订截图和不可读链接。

部署

静态输出默认适配 GitHub Pages 和 Vercel。Supabase 仅在需要账号、保存行程或协作时再接入。

本地生成命令

python skills/travel-roadbook-builder/scripts/render_roadbook.py \
  skills/travel-roadbook-builder/examples/paris-roadbook.json \
  --out docs/generated/paris-demo.html \
  --css ../assets/roadbook.css
