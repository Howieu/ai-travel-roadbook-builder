# Itinerary Rules

Use these V0 rules:

- Arrival and departure days should be lighter than full sightseeing days.
- Reserve airport and station buffers when transport time is fixed.
- Use lodging as the daily anchor when available.
- Do not place too many must-go stops on the same day.
- Add fallback notes for packed days.
- Respect avoid places.
- Prefer grouping stops by neighborhood when obvious from context.
- Do not claim exact live transport or opening-hour accuracy unless the user provided verified information.
- For a relaxed pace, use 2-4 major stops per day.
- For a standard pace, use 4-6 stops per day.
- For a packed pace, use 6+ stops only when transport risk is low.

When the input is incomplete, generate a useful draft and clearly mark assumptions.
